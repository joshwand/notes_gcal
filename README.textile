h1. notes_gcal.rb

Sync your Lotus Notes Calendar to Google Calendar (one-way sync).

h2. Homepage

http://github.com/joshwand/notes_gcal

h2. Installation

<pre>
$ gem install bundler
$ bundle install
$ cp config.yml.example config.yml
# enter your Lotus Notes and Google Account info
$ ruby notes_gcal.rb
</pre>
